## Overview

Task Management Web Application is a simple, no-dependency web app that lets users create, track, and manage tasks directly in the browser. Built as a frontend-only project — no frameworks or build tools required.

## Features

- Add, edit, and delete tasks
- Mark tasks as complete or pending
- Responsive and clean UI
- No dependencies — runs directly in the browser

## Getting Started

No installation or build steps needed.

```bash
# Clone the repository
git clone https://github.com/himasree-d/Task-Management-Web-Application.git

# Open in browser
open index.html
```

## Usage

1. Open `index.html` in any modern web browser
2. Enter a task name and submit
3. Check off tasks as you complete them
4. Delete tasks you no longer need

- HTML5
